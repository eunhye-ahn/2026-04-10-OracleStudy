# SQL 내장함수 정리

## 문자열 내장함수
- `length` : 문자 개수
- `substr` : 문자 자르기
- `instr` : 문자번호 검색
- `rpad` : 문자 채우는 함수
- `replace` : 문자를 변경

## 숫자 내장함수
- `mod` : 나머지
- `ceil` : 올림 - 정수로 처리
- `trunc` : 내림
- `round` : 반올림

## 날짜 내장함수
- `sysdate` : 오늘날짜
- `months_between` : 두 날짜의 개월의 차

## 변환함수
- `to_char` : 숫자 날짜 => 문자열 변환

### 날짜, 숫자 변환
- 숫자 : `1,000` => `9,999,999`
- 년도 : `YY` / `YYYY`
- 월 : `MM`
- 일 : `DD`
- 요일 : `DAY`
- 시 : `HH` / `HH24`
- 분 : `MI`

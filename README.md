# SQL 내장함수 정리

## 문자열 내장함수

- `length` : 문자 개수
- `substr(문자열, 시작위치, 자를개수)` : 문자 자르기
- `instr(문자열, 시작위치, 몇번째)` : 문자위치번호 검색
  - 시작위치 : `indexOf`, `lastIndexOf` 로도 가능
- `rpad(문자열, 출력개수, 대체문자)` : 문자 채우는 함수
- `replace` : 문자를 변경

---

## 숫자 내장함수

- `mod` : 나머지
- `ceil` : 올림 - 정수로 처리 → 총페이지
- `trunc` : 내림
- `round` : 반올림

---

## 날짜 내장함수

- `sysdate` : 오늘날짜
- `months_between` : 두 날짜의 개월 차
- `add_months` : 개월 등록 → 며칠인지

---

## 변환함수

- `to_char` : 숫자 or 날짜 → 문자열 변환
- `to_date(문자열, 형식지정)` : 문자열을 날짜로 변환 시, 형식을 지정하여 변환 가능

> 날짜 계산 시에는 date 객체로 변환 후 연산 → `+`, `-` 로 제어 가능

### 날짜 · 숫자 포맷

| 구분 | 포맷 |
|------|------|
| 숫자 | `9,999,999` |
| 년도 | `YY` / `YYYY` |
| 월 | `MM` |
| 일 | `DD` |
| 요일 | `DAY` |
| 시 | `HH` / `HH24` |
| 분 | `MI` |

---

## 기타 함수

- `nvl(칼럼, 값)` : null 값이 있는 경우 다른 값으로 변경
- `decode` : `=` 연산만 가능한 셀렉트문
- `case` : 가독성이 좋고 모든 연산이 가능한 셀렉트문
- `rank` / `dense_rank` : 순위

---

# 서브쿼리

- **중첩 서브쿼리** : 일반적인 서브쿼리 중첩 사용
- **스칼라 서브쿼리** : JOIN 대체 → `SELECT` 뒤에 위치
- **인라인 뷰** : 한 번만 사용 (보안, 페이징 등) → `FROM` 뒤에 위치

# ORACLE 페이징

- **rownum을 이용한 페이징**
  - rownum은 미리 모든 페이지를 저장해두고 자르는 방식

- **offset을 이용한 페이징**
  - 최근 방식의 페이징
  - 앞의 행을 건너뛰고 다음 행 출력
  - `offset = (페이지번호 - 1) × 페이지당 행수`
